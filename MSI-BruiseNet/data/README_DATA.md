# Data Directory — MSI-BruiseNet

> **This directory is where you place your own multispectral imaging data.**
> The actual `.npy` data files are excluded from version control via `.gitignore`.

## Directory Structure

```
data/
├── images/          # MSI image arrays
│   ├── sample_001.npy
│   ├── sample_002.npy
│   └── ...
├── masks/           # Binary segmentation masks
│   ├── sample_001.npy
│   ├── sample_002.npy
│   └── ...
├── splits/          # Auto-generated fold indices
│   └── splits.json  # Created by scripts/prepare_splits.py
├── norm_stats.json  # Auto-generated by scripts/compute_norm_stats.py
└── README_DATA.md   # This file
```

## Image Format

| Property | Value |
|----------|-------|
| Format   | `.npy` (NumPy binary) |
| Shape    | `(H, W, 9)` — height x width x 9 spectral bands |
| dtype    | `float32` or `uint16` |
| Bands    | 713, 736, 759, 782, 805, 828, 851, 874, 897, 920 nm |

## Mask Format

| Property | Value |
|----------|-------|
| Format   | `.npy` (NumPy binary) |
| Shape    | `(H, W)` — same H, W as the corresponding image |
| dtype    | `uint8` |
| Values   | `0` = background, `1` = bruise |

## Naming Convention

- Image and mask files must share the **exact same name**.
- Recommended pattern: `sample_001.npy`, `sample_002.npy`, ...
- The name (without `.npy`) is used as the sample identifier throughout the pipeline.

## Quick Check

```python
import numpy as np

img = np.load("data/images/sample_001.npy")
mask = np.load("data/masks/sample_001.npy")

print(img.shape)   # Expected: (H, W, 9)
print(img.dtype)   # Expected: float32 or uint16
print(mask.shape)  # Expected: (H, W)
print(mask.dtype)  # Expected: uint8
print(np.unique(mask))  # Expected: [0, 1]
```
